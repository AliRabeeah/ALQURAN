# تطبيق القرآن الكريم بصوت الشيخ محمد صديق المنشاوي

تطبيق أندرويد يعرض السور الـ 114 كاملة، مع إمكانية تحميل كل سورة مرة واحدة
ثم الاستماع إليها بدون إنترنت لاحقًا (تشغيل محلي من داخل الهاتف).

## كيفية فتح المشروع
1. افتح **Android Studio**.
2. اختر **Open** ثم حدد مجلد `QuranMinshawi`.
3. انتظر حتى ينتهي Gradle من المزامنة (Sync) — سيحمّل التبعيات المطلوبة تلقائيًا.
4. وصّل هاتفك عبر USB (مع تفعيل وضع المطور و USB Debugging) أو استخدم محاكي (Emulator).
5. اضغط **Run ▶** لتثبيت التطبيق وتشغيله.

## المميزات
- قائمة بجميع السور الـ 114 بأسمائها.
- زر "تحميل" بجانب كل سورة، وزر "تحميل الكل" أعلى الشاشة لتحميل كل ما تبقى دفعة واحدة.
- بعد تحميل السورة، تُشغَّل من الملف المحفوظ محليًا داخل تخزين التطبيق الخاص، دون الحاجة لإنترنت مرة أخرى.
- مشغل مصغّر أسفل الشاشة مع زر تشغيل/إيقاف وشريط تقدّم، ويعمل في الخلفية عبر إشعار.
- عند انتهاء سورة أثناء التشغيل، ينتقل تلقائيًا للسورة التالية إذا كانت محمّلة.

## مصدر الملفات الصوتية
يستخدم التطبيق روابط التحميل المباشرة العامة الخاصة بموقع mp3quran.net
(القارئ: محمد صديق المنشاوي - رواية حفص عن عاصم - مرتل)، وهي روابط متاحة
للجميع من خلال الموقع نفسه. يحتاج التطبيق إنترنت فقط عند تحميل كل سورة
لأول مرة، وبعدها تعمل بدون إنترنت.

إن رغبت باستخدام قارئ آخر أو مصدر صوتي مختلف، يمكن تعديل قيمة
`downloadUrl` في ملف `Surah.kt` بسهولة.

## هيكل المشروع
```
app/src/main/java/com/minshawi/quran/
├── MainActivity.kt        // الشاشة الرئيسية وقائمة السور
├── Surah.kt                // بيانات السور وروابط التحميل
├── StorageHelper.kt         // إدارة الملفات المحلية
├── DownloadHelper.kt         // تحميل الملفات عبر DownloadManager
├── PlaybackService.kt        // خدمة التشغيل في الخلفية
└── SurahAdapter.kt            // محول قائمة السور
```
